MystreamHub Tutorial

This is how to use the android app to watch your favorites videos.

Press the button new directory (1) to add new remote directory using its url.

Press the button import/export (2) to save your directories database or import a database of directories using url.

Press search button (3) to search inside your database for your movies or series to watch directly by url.

Put directory url in to text editor place (1) to start searching inside the url.

Press the button scan (2) to grab all video files direct urls from the remote directory url inserted in (1) and wait for the operation

Press save button (3) to pin directory in the home page for easy access next time.

After pressing on directory at home page you are served here after playlist generated press play.
